# hyrule

View issues offline from a public GitHub repository.

## Requirements

- Node 4.2.x

## Installation

```bash
$ npm install -g hyrule
```

## Development

1. Clone this repository

Then run

```bash
$ npm install
```

After that finished, run

```bash
$ npm link
```

Then you're able to start the app just by running

```bash
$ hyrule
```

in your terminal.
